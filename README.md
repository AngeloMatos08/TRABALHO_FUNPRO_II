# TRABALHO_FUNPRO_II
Este projeto consiste na criação de uma página inicial (homepage) com conteúdo informativo, utilizando conceitos básicos de HTML trabalhados em sala. O objetivo é praticar a estruturação de páginas web e o uso de elementos semânticos.
Neste trabalho foram implementadas ao HTML:
-Estrutura básica da página com título, cabeçalho e parágrafos;
-Links e navegação dentro da própria página (botão que retorna ao topo por meio do id);
-Imagens dentro da estrutura HTML;
-Tabela para contatos;
-Formulário com envio para servidor de testes;
-Simulação de testes erro/sucesso.

Como testar o formulário:
basta usar o usuário com "admin" e a senha: "1234".

A parte mais difícil foi a de divisão da página em sections pois eu não sabia muito bem em como refazer a página depois, porquê até aquele momento ela havia sido dividida apenas com uma linha horizontal;
A parte mais fácil acredito que tenha sido a de criar a estrutura base da página e enviar as imagens.

Também achei muito interessante a simulação do sucesso e erro do "login", gosto muito da lógica por trás.
